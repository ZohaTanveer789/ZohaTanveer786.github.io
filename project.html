Personal Portfolio Website

Project Overview
This is a personal portfolio website that I built and deployed using GitHub Pages. The purpose of this project is to create an online presence that showcases who I am, what I'm learning. It acts as my digital resume — easily shareable with recruiters, employers, or collaborators.

Video Editing

Project Overview
From 2020 to 2022, I worked as a content editor for an individual creator, where I edited videos and digital content according to requirements. My role involved trimming clips, arranging sequences, improving flow, adding basic effects, and making sure the final output matched the intended message. This experience helped me develop an eye for detail, storytelling, and working with feedback under deadlines.

About the Project

  • Followed client instructions and incorporated feedback
  • Managed multiple edits over time with consistency

Key Skills Developed
Video Editing Basics — Trimming, sequencing, transitions.
Storytelling — Enhancing narrative flow.
Attention to Detail — Ensuring quality output.
Client Communication — Understanding and implementing feedback.
